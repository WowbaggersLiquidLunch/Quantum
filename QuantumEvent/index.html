Class Quantum​Event 

public final class QuantumEvent<State: Hashable>

An instantaneous event that might have happened and moved a quantum mechanical system from one state to another state.

Currently interlacing of observed and unobserved events is not supported. An event tree must start with an observed event, followed by an uninterrupted chain of observed events, until the first unobserved event. No observed event can follow from an unobserved event.
